HEAVE is a documentation-first engineering repository for a premium AI operating workspace. It is designed to prepare a future implementation team, not to ship product code. The repository is organized so architecture, prompts, brand rules, UI references, database guidance, testing guidance, and delivery workflow can be reviewed in a stable order.
